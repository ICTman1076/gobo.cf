## What is Gobo.cf?

Gobo.cf is a URL Shortener For Scratch Projects started by [@Thermodynamical](https://scratch.mit.edu/users/Thermodynamical)

## Shortened URLs

| URL | Function          |
| --- | ----------------- |
| /u# | To User Page      |
| /s# | To Studio Page    |
| /p# | To Project Page   |
| /e# | To Project Editor |
| /d# | To Topic          |
